故障处理

      
    
        
  • 接口新增失败：检查 funcNo + sysId 是否重复。
    • 
        
  • 案例新增失败：确认功能号已先在接口表中创建。
    • 
        
  • hisId 查询失败：确认任务是否存在。
    •
